npm install
npm run start

## 기능

1. 상태관리는 Redux를 이용
2. 모달을 이용해 옵션을 선택할 수 있음
3. 던지기, 다시하기, 옵션 선택 버튼이 있어 언제든지 자유롭게 선택 할 수 있음
4. 게임에서 지거나, 이기게 되면 기존 Text가 변경되어 승자 및 패자를 알려줄 수 있음
