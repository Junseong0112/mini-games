# 틱택토

> 두 명이 번갈아가며 "O" 와 "X" 같은 글자를 3 X 3 가로, 세로, 혹은 대각선에 3개를 잇는 게임.

![main-project](https://github.com/Junseong0112/mini-games/assets/108931006/ef084959-6630-4ee1-91a3-231c70e4357a)

## 실행

```shell
npm install
npm run dev
```
